## üöÄ Quick Start / Installation

To get started with **Himura TokyoNight**, ensure you are on an Arch-based system (CachyOS, Arch, EndeavourOS) and run the following commands:

### Clone Repo
``` bash
  git clone [https://github.com/LinkoXAE/DotFile-TokyoHimura.git](https://github.com/LinkoXAE/DotFile-TokyoHimura.git)

  cd DotFile-TokyoHimura

  
  chmod +x installer.sh
  
  ./installer.sh

```
---

#### üõ† What happens next?

> The installer will automatically:

    - Install system dependencies (Python, PyQt6, Waybar, etc.).

    - Set up a local Virtual Environment (.venv).

    - Launch the TokyoNight GUI where you can review keybinds and start the config deployment.
